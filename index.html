Project Highlight – ISS Data ETL Pipeline
ISS Real‑Time Location Tracker (NASA Open APIs)

Automated ETL pipeline that captures, converts, and stores ISS coordinates for historical analysis and visualization.

⌄

I developed a fully automated ETL pipeline that captures, processes, and stores real‑time positional data of the International Space Station (ISS) using NASA's open API ecosystem.

Key Features

  • Real‑Time Data Capture: Fetches the live ISS coordinates every 5 minutes
  • Automated Timestamp Conversion: Converts UTC timestamps to IST
  • Data Storage for Analytics: Appends processed data to a CSV for a historical dataset
  • ETL Automation: Orchestrated end‑to‑end using Apache Airflow
  • Lightweight Execution Layer: Python transforms with a Bash wrapper

Tech Stack: Apache Airflow | Python | Bash | NASA Open APIs | CSV Storage

Purpose: Enables long‑term analysis and visualization of the ISS trajectory—useful for astronomy enthusiasts, data science exploration, and real‑time space observation dashboards.
